  • Main IDE window and syntax-highlighted file window
      Project files in expandable categories on left, settings on right, and messages on the bottom

  • Place the cursor on any keyword and hit F1 to bring up on-line help (STR example)
  • Double-click on error and warning messages to open up the source file and highlight the line
  • Source-level simulation highlights the next line to be executed
  • Disassembly-style simulation highlights the next address to be executed
    • Double-click on a disassembly line to open up the source file and highlight the line
    • You can use source and disassembly simulation simultaneously

  • Right-click on source or disassembly to pop-up a context-sensitive simulation dialog
    • Simulate to the selected line/instruction
    • View contents of selected data/registers/memory
    • Set instruction, data, memory, and register breakpoints
    • Perform edit and continue operations

  • A selection of some of the dialogs you can use to view and edit the state of simulation
